# pyMyBandeco
A scraper to read the menu of the Unicamp's Restaurant

## How to use pyMyBandeco

- Clone the repo:

$ git clone https://github.com/Bruno-Eduardo/pyMyBandeco.git

- Run the script

$ ./scraper.py

## Requirements
python 3.6.9 or greater

pip3

### pip modules
beautifulsoup4 >= 4.6.0

urllib3 >= 1.25.3
